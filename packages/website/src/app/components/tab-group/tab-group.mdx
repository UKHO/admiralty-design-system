# Tab group
To allow users to open different sections within the current one, with content grouped to a relevant tab heading.

## Basic
Use the tab group component to allow users to navigate to different sections within the current section.

import Basic from "@/usage/tab-group/basic/index.mdx"

## With complex content
The complex content tab group helps to organise content into a more structured format

## With selected index
Selected index can be used to select the tab (and therefore tab content) to show.

## Accessibility
This component and its versions, support the user in navigating through complex content. Ensure labelling is clear. The tabs and tab content are navigable via the keyboard.
