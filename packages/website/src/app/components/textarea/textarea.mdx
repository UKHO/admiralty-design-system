# Textarea
Use a textarea when the user is expected to enter more than a single line of text.

## Basic
The textarea component should be used for large amounts of text. Use other form elements for smaller amounts.

## Disabled
Use the disabled component when the design calls for it.

## Validation
Use validation messages to inform users that an action is required, for example entering text into the field.

## With text
Text can be pre-populated into the textarea.

## Accessibility
Users are able to enter a large amount of text into this component, it accommodates keyboard navigation and is labelled correctly.
