# Tooltips
Use the tooltip when the user is requiring further information about a specific element on the page. [View this component on GitHub](https://github.com/UKHO/admiralty-design-system/tree/main/packages/core/src/components/tooltip)

## Basic
The tooltip component should be used to provide brief, supplemental, and non-critical information about a specific UI element when a user hovers over or focuses on it.
