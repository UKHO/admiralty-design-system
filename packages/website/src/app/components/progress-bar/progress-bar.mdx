# Progress bar
Use the progress bar to signal progress to users.

## Some progress
Use the progress bar and its various states to signal to users that the system is doing something. Make sure that the component has an ARIA label to explain the context.

## No progress
Use the progress bar and its various states to signal to users that the system is doing something.

## Maximum progress
Use the progress bar and its various states to signal to users that the system is doing something.

## Error
Use the error state to signal to users that something has gone wrong.

## Labelled
Use a visual label when appropriate.

## Accessibility
The component can be given an accessible label that describes the element. As the component is a visual signifier, care should be taken to explain to visually impaired users what progress is being indicated, for example the progress of their download.
