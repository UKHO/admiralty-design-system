# Footer
Use to provide users with consistent, useful links as well as trust and reassurance. [View this component on GitHub](https://github.com/UKHO/admiralty-design-system/tree/main/packages/core/src/components/footer)

## Contents

## Basic
Our footer should be used at the bottom of a website to clearly see the UK Hydrographic Office logo and copyright information and accessibility statement. It has a set width and is responsive.

import Basic from "@/usage/footer/basic/index.mdx";

<Basic />

## With links
Footer with links provides the capability to add links to the footer. These can be used for links to accessibility statements, privacy policies etc.

import WithLinks from "@/usage/footer/with-links/index.mdx"

<WithLinks />

## Variant compact
Minified footer for interfaces which require majority of the view.
