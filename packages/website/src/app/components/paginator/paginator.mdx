# Paginator
Use to reduce overwhelming lists and make long tables easy to review.

## Contents

## Basic

The paginator component helps users view a large list or set of results. The contents can be put into chunks as determined by your designs, enabling users to gradually review a table for example.

## No label

To be used when a label is not required, with buttons only.

## Accessibility

This component is labelled for screen readers and works with keyboard navigation. It is helpful for users as a way of breaking up lengthy content in a table.
