| Name                         | Type                                                                 | Default     | Description                                                     |
| ---------------------------- | -------------------------------------------------------------------- | ----------- | --------------------------------------------------------------- |
| `allow-back-navigation`      | `boolean`                                                            | `true`      | Whether navigation to previous steps is allowed                 |
| `allow-forward-navigation`   | `boolean`                                                            | `false`     | Whether navigation to future steps is allowed                   |
| `validate-before-navigation` | `boolean`                                                            | `false`     | Whether to validate the current step before allowing navigation |
| `validate-step`              | `(stepId: string, stepIndex: number) => boolean \| Promise<boolean>` | `undefined` | Function to validate a step (returns true if valid)             |
