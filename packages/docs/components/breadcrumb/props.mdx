
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `active` | `boolean` | `false` | When `true` the breadcrumb will by styled to show that it is the currently active breadcrumb. Defaults to `true` for the last breadcrumb if it is not set on any. |
| `href` | `string` | `undefined` | Contains a URL or a URL fragment that the hyperlink points to. |
