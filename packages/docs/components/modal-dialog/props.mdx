
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `description` | `string` | `undefined` | Describe the contents of the dialog. |
| `heading` | `string` | `undefined` | The title of the modal dialog. |
| `label` | `string` | `undefined` | Label the dialog. |
| `show` | `boolean` | `false` | Whether to show the modal dialog. |
