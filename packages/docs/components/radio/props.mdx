
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `checked` | `boolean` | `false` | Determines whether the radio button is selected (or checked) |
| `disabled` | `boolean` | `false` | Determines whether the radio button is disabled. A button in disabled state will not fire click output events. |
| `invalid` | `boolean` | `false` | Determines whether to add the invalid stying to the radio button |
| `name` | `string` | `this.inputId` | The name of the radio button for use on selection within a radio group |
| `value` | `string` | `undefined` | The value of the radio button for use on selection within a radio group |
