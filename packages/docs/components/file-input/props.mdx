
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `invalid` | `boolean` | `false` | Whether to show that the file input is in an invalid state. |
| `invalid-message` | `string` | `null` | The message to show when the file input is invalid. |
| `label` | `string` | `'Click to choose a file or drag it'` | Used to display instructions to the user and is replaced with the filename the user inputs |
| `multiple` | `boolean` | `false` | If true, enables multiple files to be selected or dragged |
