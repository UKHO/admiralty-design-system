
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `display-vertical` | `boolean` | `false` | Setting this true displays the radio options vertically (defaults to false) |
| `invalid` | `boolean` | `false` | Whether to show the input in an invalid state |
| `invalid-message` | `string` | `undefined` | The message to show when the input is invalid |
| `name` | `string` | `this.inputId` | The name of the control, which is submitted with the form data |
| `value` | `any` | `undefined` | The value of the radio group |
