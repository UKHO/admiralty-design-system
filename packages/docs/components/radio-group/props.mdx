
  | Name | Type | Default | Description |
| --- | --- | --- | --- |
| `disabled` | `boolean` | `false` | Whether the radio controls should be disabled |
| `display-vertical` | `boolean` | `true` | Setting this to false displays the radio options horizontally (defaults to true) |
| `hint` | `string` | `undefined` | The hint text to display below the label |
| `invalid` | `boolean` | `false` | Whether to show the input in an invalid state |
| `invalid-message` | `string` | `undefined` | The message to show when the input is invalid |
| `label` | `string` | `undefined` | The label text to display above the control |
| `name` | `string` | `this.inputId` | The name of the control, which is submitted with the form data |
| `value` | `any` | `undefined` | The value of the radio group |
